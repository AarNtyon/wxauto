# 微信公众号自动化发布 Skill 设计文档

日期: 2026-03-23

## 概述

一个完整的微信公众号文章自动化发布 skill，从主题输入到草稿箱发布，采用分步确认模式，每个阶段都向用户展示结果并获取确认后再继续。

## 核心功能流程

```
┌─────────────────┐
│  1. 用户提出主题   │ ◄── 触发点："帮我写一篇关于...的公众号文章"
└────────┬────────┘
         ▼
┌─────────────────┐
│  2. Brave 搜索   │ ◄── 收集素材（展示搜索结果给用户确认）
│    获取素材      │
└────────┬────────┘
         ▼
┌─────────────────┐
│  3. AI 写作      │ ◄── 调用写作风格 skill 生成文章（展示给用户确认）
└────────┬────────┘
         ▼
┌─────────────────┐
│  4. 封面生成     │ ◄── 调用豆包 API（让用户选择风格，展示预览确认）
└────────┬────────┘
         ▼
┌─────────────────┐
│  5. 格式编排     │ ◄── 转换为公众号格式（展示排版效果确认）
└────────┬────────┘
         ▼
┌─────────────────┐
│  6. 发布草稿箱   │ ◄── 调用公众号 API 发布（确认后执行）
└─────────────────┘
```

## 触发词

## 配置管理

首次安装 skill 时，引导用户完成配置：
- `WECHAT_APPID` - 微信公众号 appid
- `WECHAT_SECRET` - 微信公众号 secret
- `DOUBAO_API_KEY` - 豆包 API key（用于封面生成）
- `BRAVE_API_KEY` - Brave Search API key（用于搜索）

配置存储在 skill 目录下的 `config.json` 中。

## 模块设计

### 1. 搜索模块 (search.py)
- 调用 Brave Search API
- 根据主题搜索相关素材
- 返回结构化搜索结果（标题、摘要、链接）
- 支持结果过滤和摘要

### 2. 写作模块 (writer.py)
- 调用写作风格 skill 生成文章
- 接收搜索结果作为上下文
- 生成符合公众号风格的文章内容
- 输出：标题、正文、摘要

### 3. 封面生成模块 (cover.py)
- 调用豆包 API 生成封面图
- 支持多种风格选择：
  - 写实风格
  - 插画/卡通风格
  - 极简/扁平化
  - 其他自定义风格
- 封面尺寸：900×383（首图）或 500×500（缩略图）

### 4. 格式编排模块 (formatter.py)
- 将 Markdown 转换为公众号 HTML
- 处理公众号特定样式
- 支持代码高亮、引用、列表等格式
- 输出可直接用于发布的富文本

### 5. 发布模块 (publisher.py)
- 调用微信公众号 API
- 获取 access_token
- 上传封面图片到微信素材库
- 创建图文消息草稿
- 返回草稿链接

## 交互流程

每个步骤的交互模式：

1. **执行步骤** → 2. **展示结果** → 3. **等待用户确认** → 4. **继续或修改**

用户可以在每个步骤选择：
- ✅ 确认并继续
- 🔄 重新生成/修改
- ⏭️ 跳过此步骤
- ❌ 终止流程

## 错误处理

- API 调用失败时自动重试（最多 3 次）
- 配置缺失时提示用户补全
- 每个步骤的错误独立处理，不影响其他步骤

## 依赖
